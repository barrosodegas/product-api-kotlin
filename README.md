
# CRUD API Kotlin with Spring Boot example.

### Endpointes:

[ ] GET - Find product(s)

- http://localhost:8180/products

- http://localhost:8180/products/2/id

- http://localhost:8180/products/12345/sku

[ ] POST - Create product

- http://localhost:8180/products

[ ] PUT - Update product

- http://localhost:8180/products

[ ] DELETE - Remove product

- http://localhost:8180/products/12345/id

- http://localhost:8180/products/12345/sku
